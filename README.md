<h2>🛠️ Installation Steps:</h2>

<p>1. Install Packages</p>

```
npm install
```

<p>2. Run The Scrap</p>

```
npm run scrap
```

<p>3. Run Data process [Run this after Scrap Finished Running]</p>

```
npm run pros
```
